Backup & Restore

Export — save a backup

Tap the button below to download a backup file of all your season data. Save it to your iPhone's Files app or iCloud.

Restore — load a backup

Select a previously exported backup file to restore your data. This will replace all current data.

Backup tips

• Export a backup after each game day.

• The file will be saved to your Downloads folder in the Files app.

• Updating the app on GitHub will never delete your data.

Reset — clear all data

This will permanently delete all players, rotations, awards, duties and season statistics. This cannot be undone. Export a backup first if you want to keep a copy.
